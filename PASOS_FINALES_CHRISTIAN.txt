═══════════════════════════════════════════════════════════════════

1. Ve a: https://dashboard.render.com
2. Click en "New +" → "Web Service"
3. Click en "Build and deploy from a Git repository"
4. Selecciona tu repositorio "Portafolio"
5. Click en "Connect"

CONFIGURACIÓN:

   Region: Oregon (US West)
   Branch: main
   Root Directory: server
   Runtime: Node
   Build Command: npm install
   Start Command: npm start

VARIABLES DE ENTORNO (Environment Variables):
   EMAIL_USER = christianmanuel1233@gmail.com
   EMAIL_PASS = [contraseña de aplicación de Gmail - 16 caracteres]
   PORT = 5000

6. Click en "Create Web Service"
7. Espera 3-5 minutos
8. COPIA LA URL (será algo como):
   https://christian-estrada-backend.onrender.com
