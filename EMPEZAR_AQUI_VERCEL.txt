═══════════════════════════════════════════════════════════════════

PASO 1: Preparar (5 min)
   • Verificar que compile: npm run build
   • Crear cuentas: GitHub, Vercel, Render

PASO 2: GitHub (5 min)
   • Crear repositorio en github.com
   • Subir tu código con Git

PASO 3: Vercel - Frontend (3 min)
   • Importar repositorio en vercel.com
   • Deploy automático
   • Obtener URL: https://tu-portfolio.vercel.app

PASO 4: Render - Backend (5 min)
   • Crear Web Service en render.com
   • Configurar carpeta: server
   • Agregar variables: EMAIL_USER, EMAIL_PASS

PASO 5: Conectar (3 min)
   • Actualizar URL en Contact.jsx
   • Push a GitHub
   • Vercel redespliega automático

PASO 6: Probar (2 min)
   • Verificar que todas las páginas funcionen
   • Probar formulario de contacto
